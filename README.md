# Keyhub Translation Tool (KeyHub-tt)

## Description
Translation tool for key-hub.eu for translators.

## How To Use

- Open the [**website**]().
- Edit the given strings.
- Export it and send a PR to [**here**](https://github.com/jondycz/KeyHub).
